**Block all popups**

- Go to `about:config`.
- Change `dom.popup_allowed_events` to empty.


**Always open popup in new tab**

- Go to `about:config`.
- Change `browser.link.open_newwindow.restriction` to 0.
- Make sure `browser.link.open_newwindow` is set to 3 (default value).
